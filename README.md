# Tailwind CSS ვებ გვერდის სტილიზაცია - გამოცდა

## 📋 პროექტის აღწერა

- **Tailwind CSS** ფრეიმვორკის პრაქტიკული გამოყენება
- **Responsive დიზაინი** (მობილური, დესკტოპი)
- **Utility-first** მიდგომის გამოყენება
